# UrbisMap QGIS Plugin

Plugin QGIS che permette di aprire UrbisMap direttamente nel punto cliccato sulla mappa.

## Funzionamento
- Attiva il plugin
- Clicca sulla mappa
- Si apre automaticamente UrbisMap nel browser sulla posizione selezionata

## Requisiti
- QGIS 3.x
